## Prerequirements

```sh
$ cat .nvmrc
```

## Setup

```sh
$ npm i
```

## Start dev-server

```sh
$ npm run dev
```

## Build assets

```sh
$ npm run build
```

## Deploy

Merge to a default branch will trigger GitHub Actions that deploys assets to Cloudflare Pages.
